# VOG PruebasAL

Extensión de prueba para Business Central siguiendo los estándares AL de VOG. Incluye una extensión de página inicial y configuración base para AppSourceCop.

**Rangos de ID:** 50650 - 50699

 * [Descripción Objetos](#descripción-de-objetos)
    * [Extensiones Paginas](#extensiones-paginas)
* [Requisitos Previos](#requisitos-previos)
* [Uso Funcional](#uso-funcional)

## Descripción de Objetos

### Extensiones Paginas

| ID | Nombre | Descripción |
| ------| -----| --- |
| 50650 | VOGCustomerList | Extiende "Customer List" para mostrar un mensaje informativo al abrir la página. |

## Requisitos Previos

Business Central 27.0.0.0 y permisos para publicar extensiones.

## Uso Funcional

Abrir la página "Customer List"; al cargar, se muestra un mensaje informativo confirmando la publicación de la extensión. Esta lógica es temporal y puede reemplazarse por funcionalidad real según el módulo.
